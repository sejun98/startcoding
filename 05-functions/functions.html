함수, 일련의 코드 묶음!

함수, 무엇?

  • 일련의 코드 묶음 ➡ 코드 재사용성 증가 🔺
  • 입력에 따른 결과 반환! (like a 자판기)
  • 함수 호출 시, 실행 흐름을 함수 내부로 이동
  • 함수 종료 시, 실행 흐름은 이전 위치로 되돌아감

파라미터, 입력값을 저장하는 변수!

  • 필요에 따라 개수 조절 가능
  • 다음 코드가 동작하도록, 함수를 만들어보자!
        const ramen = createRamen();
        console.log(ramen);
        const mandoo = createMandoo("고기");
        console.log(mandoo);
        const kimbob = createKimbob("참치", "김치");
        console.log(kimbob);

return, 함수 종료와 동시에 값을 반환!

    function NAME(PARAMETERS) {
        ..
        // return undefined;
    }

함수 실행 시, 변수의 활동 범위를 주의!

  • 프로그램은 실행 흐름에 따라 변수 관리함
  • 함수 호출 👉 파라미터와 지역변수를 만듦
  • 함수 종료 👉 파라미터와 지역변수를 제거
  • 지역변수란, 함수 또는 블럭(중괄호) 속에 선언된 변수
