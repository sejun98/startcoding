객체, 내가 만든 데이터!

1. 객체 만들기

    겜방에 간 홍팍.
    신성한 게임을 위해, 먼저 메뉴판을 본다.
    == 메뉴판 ==
    뽀글이(3000원)
    슈넬치킨(5000원)
    코카콜라(1000원)
    아메리카노(1500원)

2. 객체 값 읽고 쓰기

    주문을 하다보니 왠걸,
    메뉴 가격이 500원 씩 올랐다.
    == 메뉴판(가격 인상) ==
    뽀글이(3500원)
    슈넬치킨(5500원)
    코카콜라(1500원)
    아메리카노(2000원)

3. 객체 프로퍼티 확인하기

    메뉴 주문을 마치고, 게임에 접속한 홍팍.
    == 로그인 성공 ==
    name: 홍팍
    level: 40
    hp: 1200
    mp: 800
    skills: ["에너지볼트", "파이어볼", "토네이도"]

4. 객체 배열 생성 및 순회

    즐겜을 마치고, 겜방을 나서는 홍팍.
    영수증을 확인하는데..
    == 이용내역 ==
    겜방 3시간(3000원)
    슈넬치킨(5500원)
    코카콜라(1500원)
    아메리카노(2000원)
    == 총합 ==
    12000원
