조건문, 상황별 다른 실행!

소개팅#1, 라떼와 아아 중 택1

    if (맘에_드는가) {
        라떼 주문 // 맘에 드는 경우
    } else {
        아아 주문 // 맘에 안드는 경우
    }

소개팅#2: 아직 모르겠다면 카푸치노

    if (맘에_드는가) {
        라떼 주문    // 맘에 드는 경우
    } else if (아직_모르겠는가) {
        카푸치노 주문 // 아직 모르겠는 경우
    } else {
        아아 주문    // 부합하는게 없는 경우 
    }

버스요금: 나이에 따른 가격 설정

    if (18세_초과) {
        1200원    // 일반
    } else if (12세_초과) {
        720원     // 청소년
    } else {
        450원     // 어린이(12세 이하)
    }
